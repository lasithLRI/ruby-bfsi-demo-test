Accounts Central is an Open Banking demo that enables you to aggregate financial data using the Add Account feature and simulate direct payments through Pay Bills. The demo allows you to seamlessly explore both AISP and PISP journeys, as all login and OTP fields accept any alphanumeric input for simulation purposes.
